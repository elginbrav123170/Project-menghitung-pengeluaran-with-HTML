### 📝 Deskripsi Proyek
Proyek ini dikembangkan oleh **Elgin Brav Ekazatia**, mahasiswa Informatika UKDW, untuk membantu memantau aliran keuangan pribadi secara sistematis. Aplikasi ini memungkinkan pencatatan pengeluaran harian, bulanan, hingga tahunan agar manajemen finansial jadi lebih terukur dan tidak 'bocor' seperti *mana pool* yang habis.

### ✨ Fitur Utama
- **Harian:** Catat setiap transaksi real-time dengan kategori (makanan, gaming, edukasi, dll).
- **Bulanan:** Rekapitulasi total pengeluaran untuk melihat pola konsumsi tiap bulan.
- **Tahunan:** Laporan komprehensif untuk perencanaan budget jangka panjang.
- **Analisis Kategori:** Lihat ke mana uang paling banyak mengalir (apakah ke Top-up Dota atau buku kuliah?).
